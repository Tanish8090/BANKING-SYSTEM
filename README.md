# **Banking System (Python)**

A beginner-friendly command-line **banking application** built using Python.  
It provides **account creation, login, deposit, withdrawal, and balance check** features — all stored in memory.

---

## **Project Title**

**Banking System – Python CLI Application**

---

## **Overview**

This project implements a simple **banking system** using Python, designed for beginners to understand:

* Menu-driven programs

* Functions and modular programming

* Data storage using Python lists

* Basic authentication logic

* Input handling and validation

The banking data is stored in memory (no database), making it lightweight and easy to understand.

---

## **Features**

*  Create a new bank account

*  Login using username & 4-digit PIN

*  Deposit money

*  Withdraw money (with balance validation)

*  Check current balance

* Logout & switch user

* Simple, clear command-line interface

*  No external dependencies

## **Testing Instructions**

### **1\. Create an Account**

* Choose **Create Account**

* Enter a name and PIN  
   **Expected:** *Account created\!*

### **2\. Login**

* Enter correct username & PIN  
   **Expected:** *Logged in successfully.*

### **3\. Invalid Login**

* Enter wrong credentials  
   **Expected:** *Incorrect credentials\!\!*

### **4\. Deposit**

* Choose deposit

* Enter valid amount  
   **Expected:** Balance increases

### **5\. Withdraw**

* Withdraw \> balance → *Insufficient funds\!\!*

* Withdraw valid amount → Balance decreases

### **6\. Logout**

* Returns to main menu

### **7\. Quit**

* Program exits
